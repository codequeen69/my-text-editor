# My Text Editor

## Purpose
For this challenge I made a browser text editor. A user can type text into the lines in the browser and when the page is refreshed, the note persists. This application is also able to work offline. 

## Build With:
*HTML
*CSS
*JavaScript
*Node.js
*Express.js
*IndexedDB
*Babel
*Workbox
*Webpack
*Concurrently
